## 디자인 특징

### 1. 프리미엄 헤더
- 제목에 보라색 그라데이션 효과
- 깔끔한 타이포그래피

### 2. 상황 선택 영역
- 드롭다운과 TPO 설명 expander가 자연스럽게 배치
- CSS가 모든 정렬 처리
- 호버 시 시각적 피드백

### 3. 파일 업로더
- 점선 테두리로 드래그앤드롭 느낌
- 호버 시 보라색 하이라이트

### 4. 분석 결과
- 그라데이션 배경의 점수 컨테이너
- 큰 숫자로 강조된 점수 표시
- 카드 스타일 레이아웃

### 5. 버튼
- 그라데이션 primary 버튼
- 호버 시 약간 상승하는 효과
- 부드러운 그림자

### 6. 사이드바
- 세로 그라데이션 배경
- 흰색 텍스트로 대비

## 기술 노트

- CSS는 앱 시작 시 한 번만 로드됨
- `unsafe_allow_html=True`로 custom HTML/CSS 사용
- Streamlit의 기본 클래스명 활용 (`.stButton`, `.stSelectbox` 등)
- 반응형 디자인으로 모바일도 지원
